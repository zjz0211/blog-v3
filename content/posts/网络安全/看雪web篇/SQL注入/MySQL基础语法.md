## 1-1 MySQL初识

### 📚 什么是MySQL？

MySQL是一个广泛使用的关系型数据库管理系统（RDBMS），使用SQL（结构化查询语言）进行数据的增删改查操作。它是开源的，被广泛应用于Web应用开发中。
![](/images/20260316195558.png)
### 🏗️ 核心概念

|概念|说明|类比|
|---|---|---|
|**数据库 (Database)**|存储数据的容器|一个文件柜|
|**数据表 (Table)**|数据库中存储数据的结构化方式|文件柜中的一个抽屉|
|**字段 (Column)**|表中的列，定义数据的类型|表格的列标题|
|**行 (Row)**|表中的一条记录|表格的一行数据|
### 💡 常用只读命令

在MySQL中，以下是一些常用的只读命令（不会修改数据）：

- `SHOW DATABASES;` - 查看所有数据库
- `USE dbname;` - 切换到指定数据库
- `SHOW TABLES;` - 查看当前数据库的所有表
- `SELECT ... FROM ...;` - 查询数据
## 1-2 数据库查询

### 📚 什么是数据库？

我们之前有学习数据库，MySQL里的数据库，指的是，MySQL可以有很多个数据仓库，你可以理解为它可以创建"仓库1"，"仓库2"，"仓库3"。比如下图，就有如下四个数据库。

- **仓库1: school**（例如：用于学校管理）
- **仓库2: hospital**（例如：用于医院信息管理）
- **仓库3: shop**（例如：用于商城类数据）
- **仓库4: users**（例如：应用系统用户数据）
![](/images/20260316195625.png)
### 📚 查询Mysql里的数据库

在 MySQL 中，我们可以通过**查询语句**来获取我们需要的数据。例如，想要查看当前数据库系统中所有的数据库仓库，可以使用如下SQL命令：

SHOW DATABASES;

  
这个命令会查询并列出当前MySQL服务器里所有的数据库名。学会查询，是与数据库打交道的第一步。

## 1-6 select 1,2,3

### 📚 数据库查询语法

最基础的 `select username, password from users;` 相信大家已经学会了。
### 📚 语法

接下来介绍一个有趣的小知识：

**如果你执行** select 1,2,3; ，会发生什么？

这条语句不是查询数据表，而是让数据库生成一行，直接显示 **1**、**2** 和 **3** 这几个数字作为结果：

          +---+---+---+   
         |1   | 2  |3  |   
          +---+---+---+  

这常用于测试数据库连接或简单输出常量。即使没有 from 子句（没有指定表），MySQL 也能直接返回你写在 select 后的表达式。
## 1-7 where

### 📚 Where语法

有时候我们需要查询某个特定用户的账号信息，比如通过用户名查找该用户的所有信息。或者通过文章的id或者名字去查询的时候。都会使用where语句

什么是 where 子句？

`select * from users where username='输入的账号';`

**解释：** **where** 是 SQL 查询语句中的**条件子句**，用来指定筛选结果的**条件**。  
只有满足 **where** 条件的那些记录，才会出现在查询结果中；不满足条件的记录则被过滤掉。

上述例子含义：  
从 users 表中，**只返回 username 等于“输入的账号”**的那一条或多条记录，其他用户不会出现在结果里。
## 1-8 and

### 📚 AND语法

有时候我们不仅需要查询某个特定用户，还想同时限定满足多个条件，例如查询用户名为特定值且邮箱为某个邮箱的用户。这时我们就需要用到 **where** 和 **and** 语句来实现“并且查询”。

什么是 where 和 and 子句？

`select * from users where username='小明' and email='xiaoming@example.com';`

**解释：** **where** 子句用于指定筛选结果的**条件**。  
**and** 关键字可以**并且**指定多个条件，只有同时满足所有条件的记录，才会被选出来。
## 1-8 and

### 📚 AND语法

有时候我们不仅需要查询某个特定用户，还想同时限定满足多个条件，例如查询用户名为特定值且邮箱为某个邮箱的用户。这时我们就需要用到 **where** 和 **and** 语句来实现“并且查询”。

什么是 where 和 and 子句？

`select * from users where username='小明' and email='xiaoming@example.com';`

**解释：** **where** 子句用于指定筛选结果的**条件**。  
**and** 关键字可以**并且**指定多个条件，只有同时满足所有条件的记录，才会被选出来。

上例的含义：

只有这两个条件都成立的记录才会出现在查询结果里。
